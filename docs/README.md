# PrintFlow 🖨️

**Marketplace + system zarządzania farmami drukarek 3D**

Jak Uber Eats ale dla druku 3D. Właściciel farmy zakłada konto, konfiguruje drukarki, dostaje link do swojego sklepu. Klienci zamawiają, płacą, śledzą na żywo.
